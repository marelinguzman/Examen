Sistema e Instalación de software


El software permite administrar los recursos que necesita el sistema operativo del computador para manejar los programas y aplicaciones. El software sirve como puente para que el usuario interactúe con el hardware a través de este. Para la carrera de Computacion en primordial saber de esto, ya que esto nos ayuda a conocer mas lo componenetes de una Computadora, saber identificar de como este hecha la misma, y presentar el porque son importantes sus partes

  • Definición de energía alterna y directa
  • Unidad de medida de almacenamiento
  • Sistem on achip
  • Definición de GPU integrada y dedicada
